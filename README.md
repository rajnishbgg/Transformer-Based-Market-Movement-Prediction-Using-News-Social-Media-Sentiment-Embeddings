## 📌 Project Overview

This project focuses on predicting **stock market movement (UP or DOWN)** using **Transformer-based deep learning models** combined with **news and social media sentiment analysis**.

Financial markets are heavily influenced by public opinion, news, and social media discussions. This project extracts sentiment from textual data and combines it with historical market data to improve prediction accuracy.

The system uses **Natural Language Processing (NLP)** and **Transformer architectures** to analyze sentiment and forecast market trends.
